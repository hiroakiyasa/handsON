# GitHub Copilot Hands-on

プログラミング未経験者向けの、VS Code × GitHub Copilot 実践ハンズオン教材です。インストール確認から始め、Markdown原稿、Mermaid図、HTMLレポートの作成までを約90分で体験できます。

## 閲覧方法

`index.html` をEdge、Chrome、FirefoxなどのWebブラウザーで開いてください。ビルドや外部ライブラリのインストールは不要です。

ローカルサーバーを使う場合:

```bash
python -m http.server 8000
```

その後、`http://localhost:8000` を開きます。

## 主な機能

- 9章・約90分＋実例動画のステップ形式
- 13項目の完了チェックとブラウザー内の進捗保存
- そのまま使える実践プロンプト
- 画面資料の拡大表示
- 白を基調にした明るい学習UIと文字拡大切り替え
- PC、タブレット、スマートフォン対応
- 元教材のサンプル成果物と編集可能なDraw.ioファイル
- DRBFM、PDF構造化、Panel／INCA計測解析の実例動画ガイド
- Image 2.0で作成した7枚の導入・操作・業務フロー補足図
- SharePoint／OneDrive動画リンクの端末内保存（GitHubへ送信しません）

## 構成

```text
.
├── index.html
├── styles.css
├── script.js
├── video-config.js   # 公開可能な動画URLのみ任意設定
├── assets/images/     # 操作画面・構成図
├── assets/illustrations/ # Image 2.0の補足図
└── samples/           # ハンズオンの完成見本
```

## 注意

VS CodeおよびGitHub Copilotの画面、モデル、利用可能な機能は更新や組織設定によって異なります。業務利用では、所属組織の情報管理・ソフトウェア利用ルールを優先してください。AIが生成した内容、コード、コマンドは、採用・実行前に必ず人が確認してください。
